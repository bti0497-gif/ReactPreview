# 진단 협업 스튜디오 설치 및 업데이트 가이드

## 1. 시스템 요구사항
- **운영체제**: Windows 10 또는 Windows 11 (64비트)
- **메모리**: 4GB RAM 이상 권장
- **저장공간**: 500MB 이상 여유 공간
- **인터넷**: 초기 로그인 및 데이터 동기화를 위해 필요

## 2. 설치 방법
1. 배포된 설치 파일 (`JindanStudio-Setup-0.9.0.exe`)을 다운로드합니다.
2. 파일을 더블 클릭하여 실행합니다.
   - *참고: Windows Defender 경고가 뜰 경우 "추가 정보" -> "실행"을 클릭하세요.*
3. 설치가 자동으로 진행되며, 완료 후 바탕화면에 **"진단 협업 스튜디오"** 아이콘이 생성됩니다.
4. 앱이 자동으로 실행됩니다.

## 3. 초기 설정
1. **로그인**: 부여받은 ID와 비밀번호로 로그인합니다.
2. **데이터 동기화**:
   - 최초 실행 시 Google Drive에서 최신 데이터를 자동으로 동기화합니다.
   - 데이터 양에 따라 수 초~수 분이 소요될 수 있습니다.
   - 화면 하단 상태바에서 동기화 상태를 확인할 수 있습니다.

## 4. 데이터 저장 위치
- **로컬 데이터베이스**: `C:\Users\{사용자명}\AppData\Local\JindanStudio\database\`
- **로그 파일**: `C:\Users\{사용자명}\AppData\Local\JindanStudio\logs\`
- **백업 파일**: `C:\Users\{사용자명}\Documents\JindanStudio_Backup\`

## 5. 자동 업데이트
본 프로그램은 새로운 기능 추가나 버그 수정 시 자동으로 업데이트됩니다.

1. **업데이트 확인**: 앱 실행 시 또는 실행 중 주기적으로 새 버전을 확인합니다.
2. **다운로드**: 새 버전이 있으면 백그라운드에서 자동으로 다운로드됩니다.
3. **설치**:
   - 다운로드가 완료되면 "업데이트 설치" 알림이 뜹니다.
   - 앱을 재시작하면 새 버전으로 자동 교체됩니다.

## 6. 문제 해결
- **실행이 안 될 때**: 
  - 작업 관리자에서 `JindanStudio.exe` 프로세스를 모두 종료 후 재실행해보세요.
  - PC를 재부팅해보세요.
- **동기화 오류**:
  - 인터넷 연결을 확인하세요.
  - 방화벽이나 백신 프로그램이 앱의 통신을 차단하고 있는지 확인하세요.
- **데이터가 안 보일 때**:
  - `Ctrl + R` 키를 눌러 화면을 새로고침하세요.
  - 메뉴를 다시 클릭하여 데이터를 다시 불러오세요.
